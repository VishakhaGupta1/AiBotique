This folder is for virtual try-on demo datasets.

Structure:
- Each subfolder (e.g., sample1, sample2) should contain:
    - body.jpg (or .png): The user's body photo
    - garment.png: The garment image (preferably PNG with transparency)
    - (optional) result.png: The expected output for reference

You can add more samples by creating new subfolders and placing the images inside.

Example usage:
- Use these images to test or demo the virtual try-on feature.
- You can modify the backend to load from these files for testing.
